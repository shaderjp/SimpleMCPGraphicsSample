- `d3d12-mcp.png`: D3D12 Debug版、MCP接続、Transform変更
- `vulkan-mcp.png`: Vulkan Debug版、MCP接続、D3D12と同じTransform
- `mcp-camera.png`: D3D12 Debug版、`set_camera`実行例
- `mcp-light.png`: D3D12 Debug版、`set_light`でColorとIntensityを変更
- `mcp-transform.png`: D3D12 Debug版、`set_transform`実行例
